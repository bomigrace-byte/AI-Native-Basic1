### 🥇 사용할 무료 모델 목록과 용도

```
**1. 사용할 무료 모델 목록**
		- Google Gemini Flash
    - META LLaMA3 (Groq)
    - Claude Haiku (Anthropic)
		- Perplexity AI
		- GenSpark AI
		

**2. 각 모델별 용도 정하기**
   - Gemini Flash → 간단한 질문, 요약, 번역
   - META LLaMA3 → 빠른 응답 필요 시
   - Claude Haiku → 적확성 검증 및 문서 작업
   - Perplexity → 긴글 분석, GPT-4o/Claude 추가 사용하고 싶을 때 (하루 5회 제한)
   - GenSpark → 리서치, 보고서 제작

**3. 언제 유료 모델로 넘어갈지 기준 정하기**
   - 작업량이 많아져서 무료 제공으로 작업 마무리가 어려운 시점
   - 고정적으로 사용하는 워크플로우가 발생할 경우 
```

### 🥈 프롬프트 작성 규칙 (경량화)

- **경량화 규칙**

```
1️⃣ 인사말 제거      → "안녕하세요" 필요 없음
2️⃣ 자기소개 제거    → AI는 맥락 기억 못함
3️⃣ 목록으로 정리    → 문장 대신 - 또는 번호
4️⃣ 중복 제거        → 같은 말 두 번 하지 않기
5️⃣ 수치로 명시      → "적당히" ❌ → "500자" ✅
```

- **예시 3 : 번역 + 요약 요청**
    - **경량화 전 (무거운 프롬프트)**
        
        ```
        아래 영어 텍스트를 한국어로 번역해 주시고,
        번역한 내용을 3줄로 요약도 해주세요.
        번역할 때는 자연스러운 한국어로 해주시고,
        너무 딱딱하지 않게 부드러운 표현을 써주세요.
        요약은 핵심 내용만 담아서 간결하게 부탁드려요!
        
        [영어 텍스트 입력]
        
        → 총 89 토큰 사용
        ```
        
    - **경량화 후 (가벼운 프롬프트)**
        
        ```
        아래 텍스트를 :
        1. 자연스러운 한국어로 번역
        2. 3줄 요약
        
        [영어 텍스트 입력]
        
        → 총 22 토큰 사용 (75% 절약!)
        ```
        

### 🥉 AI 답변 검증 체크리스트

```
**AI 답변 받은 후 확인할 것:**

□ 사실과 다른 내용은 없는가?
□ 핵심 내용이 빠지지 않았는가?
□ 우리 서비스에 맞는 말투/형식인가?
□ 민감한 표현은 없는가?
```
